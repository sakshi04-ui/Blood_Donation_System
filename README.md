# Blood Donation & Emergency Locator System

A smart emergency response system that connects blood donors with recipients in urgent need. This system allows for **location-based donor search**, **automated matching**, and **SMS alerts via Twilio** to ensure a quick response in emergencies.

## üõ†Ô∏è Technologies Used
- **Frontend**: React
- **Backend**: Spring Boot
- **Database**: MongoDB
- **Maps**: OpenStreetMap
- **SMS Service**: Twilio

## üöÄ Features
- **Location-based Blood Donor Search**: Search for donors based on proximity to the recipient's location.
- **Automated Matching**: Matches donors with recipients in need, reducing response times.
- **SMS Alerts via Twilio**: Sends instant SMS alerts to donors and recipients when a match is made.
- **Real-time Location Tracking**: Uses OpenStreetMap for accurate tracking of donors and recipients.
- **Secure Data Management**: Ensures privacy and protection of user data.

## üîß Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/blood-donation-emergency-locator.git
````

2. **Frontend Setup (React):**

   * Navigate to the `frontend` directory.
   * Install dependencies:

     ```bash
     npm install
     ```
   * Run the React development server:

     ```bash
     npm start
     ```

3. **Backend Setup (Spring Boot):**

   * Set up your Spring Boot backend according to the documentation in the `backend` directory.
   * Run the Spring Boot application:

     ```bash
     mvn spring-boot:run
     ```

4. **Twilio Integration:**

   * Set up your **Twilio** account and get the API credentials.
   * Configure the Twilio API keys in the backend to send SMS notifications.
